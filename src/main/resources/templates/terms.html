APIsis

서비스 이용약관

최종 업데이트: 2024년 12월 9일

APIsis 서비스를 이용해 주셔서 감사합니다. 본 약관은 서비스 이용에 관한 기본적인 사항을 규정합니다.

제1조 (목적)

본 약관은 APIsis(이하 "서비스")가 제공하는 모든 서비스의 이용과 관련하여 서비스와 이용자의 권리, 의무 및 책임사항을 규정함을 목적으로 합니다.

제2조 (용어의 정의)

  • 서비스: APIsis가 제공하는 모든 API 서비스 및 관련 부가 서비스
  • 이용자: 본 약관에 따라 서비스를 이용하는 회원 및 비회원
  • 회원: 서비스에 가입하여 서비스를 이용하는 자
  • API 키: 서비스 이용을 위해 발급되는 고유 인증키

제3조 (약관의 효력 및 변경)

3.1 약관의 효력

본 약관은 서비스를 이용하고자 하는 모든 회원에 대하여 그 효력을 발생합니다.

3.2 약관의 변경

서비스는 필요한 경우 관련 법령을 위배하지 않는 범위에서 본 약관을 변경할 수 있으며, 변경된 약관은 서비스 내 공지사항을 통해 공지한 날로부터 7일 후에 효력이 발생합니다.

제4조 (회원가입)

4.1 가입 방법

회원가입은 OAuth2 소셜 로그인(카카오, GitHub, Google)을 통해 이루어집니다.

4.2 가입 승인

서비스는 다음 각 호에 해당하는 가입신청에 대하여는 승인을 거부할 수 있습니다:

  • 타인의 명의를 이용한 경우
  • 허위 정보를 기재한 경우
  • 서비스 이용을 제한 또는 정지 당한 이력이 있는 경우

제5조 (서비스의 제공)

5.1 제공 서비스

  • 다양한 도메인의 API 서비스
  • API 키 관리 및 사용량 통계
  • API 문서 및 놀이터 기능
  • 기타 서비스가 추가 개발하거나 제휴를 통해 제공하는 서비스

5.2 서비스 중단

서비스는 다음 각 호의 경우 서비스 제공을 일시적으로 중단할 수 있습니다:

  • 시스템 정기점검, 서버의 증설 및 교체
  • 서비스 개선을 위한 업데이트
  • 천재지변, 비상사태 등 불가피한 사유

제6조 (회원의 의무)

회원은 다음 행위를 해서는 안됩니다:

  • 허위 정보 등록 및 타인의 정보 도용
  • 서비스의 운영을 방해하는 행위
  • 과도한 API 호출로 서버에 부담을 주는 행위
  • 서비스를 통해 얻은 정보를 무단으로 복제, 배포, 상업적으로 이용하는 행위
  • 관련 법령을 위반하는 행위

제7조 (API 키 관리)

7.1 API 키 보안

회원은 본인의 API 키를 안전하게 관리할 책임이 있으며, API 키의 유출로 인한 손해는 회원 본인이 부담합니다.

7.2 API 사용 제한

서비스는 서버 안정성 및 공정한 이용을 위해 API 호출 횟수를 제한할 수 있습니다.

제8조 (서비스 이용 제한)

서비스는 회원이 본 약관의 의무를 위반하거나 서비스의 정상적인 운영을 방해한 경우, 서비스 이용을 제한하거나 중지할 수 있습니다.

제9조 (회원 탈퇴 및 자격 상실)

9.1 회원 탈퇴

회원은 언제든지 서비스 설정 페이지를 통해 탈퇴를 요청할 수 있습니다.

9.2 데이터 삭제

회원 탈퇴 시 모든 개인정보는 데이터베이스에서 즉시 완전히 삭제되며, 복구가 불가능합니다.

제10조 (면책조항)

  • 서비스는 천재지변 또는 이에 준하는 불가항력으로 인하여 서비스를 제공할 수 없는 경우에는 책임이 면제됩니다.
  • 서비스는 회원의 귀책사유로 인한 서비스 이용의 장애에 대하여 책임을 지지 않습니다.
  • 서비스는 회원이 서비스를 이용하여 기대하는 수익을 얻지 못한 것에 대하여 책임을 지지 않습니다.
  • 서비스는 회원이 서비스를 통해 얻은 정보를 이용하여 발생한 손해에 대해 책임을 지지 않습니다.

제11조 (준거법 및 관할법원)

본 약관과 서비스 이용에 관한 분쟁에 대해서는 대한민국 법을 준거법으로 합니다.

부칙
